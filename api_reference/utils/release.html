Create a dictionary for accessing frequency array.

The dictionary is keyed by the grouping combinations found in the frequency metadata array, where values are the corresponding 0-based indices for the groupings in the frequency array. For example, if the freq_meta entry [{‚Äògen_anc‚Äô: ‚Äònfe‚Äô}, {‚Äòsex‚Äô: ‚ÄòXX‚Äô}] corresponds to the 5th entry in the frequency array, the returned dictionary entry would be {‚Äònfe_XX‚Äô: 4}.
